# Android_Dynamic_Layout

## 动态布局
- 点击add增加一个控件
- 点击输入框的右侧加号移除这个布局
- 每当变化时，输出控件的id和文本框内容

## 说明
- 需要移除控件的时候，其实把这个控件“隐藏”了，在HashMap里面处理数据的移除
